# CV Application

**Live Preview:** [tarikdmrts-cv-application.vercel.app](https://tarikdmrts-cv-application.vercel.app/)


## Overview

This project is a dynamic CV builder that allows users to input their personal details, educational background, and practical experience.

### Features

- **General Information:** Add and edit personal details such as name, email, and phone number.
- **Educational Experience:** Add multiple educational entries, including school name, field of study, and dates of study.
- **Practical Experience:** Add multiple professional work experiences, including company name, job title, description, and employment dates.
